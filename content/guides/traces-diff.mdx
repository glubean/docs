# Traces & Diff

Glubean stores request/response traces so you can compare behavior across runs.

## Typical flow

1. Run a test.
2. Open the newest trace in the Trace Viewer.
3. Diff with the previous run.

In the extension, use:

- `Glubean: Open Latest Trace`
- `Glubean: Diff with Previous Run`

CLI users can still consume the same artifacts from disk. Paths are documented in [Reference: Artifacts](/reference/artifacts).

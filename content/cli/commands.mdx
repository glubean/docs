# CLI Commands

The Glubean CLI is the engine behind the VS Code extension. Install it manually if needed:

```bash
# macOS / Linux
curl -fsSL https://glubean.com/install.sh | sh

# Or with Deno
deno install -Agf jsr:@glubean/cli
```

## Commands

| Command | Description |
|---------|-------------|
| `glubean init` | Scaffold a new project in the current directory |
| `glubean run <file>` | Run tests in a file |
| `glubean run --all` | Run all tests in the workspace |
| `glubean context` | Generate an AI context file |
| `glubean setup` | Install or verify Deno and CLI |

## `glubean run`

```bash
glubean run explore/api.test.ts
glubean run explore/api.test.ts --filter get-user
glubean run explore/api.test.ts --pick admin
glubean run --all --verbose
```

## `glubean context`

Generates `.glubean/ai-context.md` — a file you can feed to any AI assistant for better test generation.

```bash
glubean context
glubean context --openapi spec.json
```

# Extension Commands & Settings

## High-use commands

- `Glubean: Run All Tests in File`
- `Glubean: Run All Tests in Workspace`
- `Glubean: Re-run Last Request`
- `Glubean: Open Latest Trace` (opens Trace Viewer)
- `Glubean: Diff with Previous Run`
- `Glubean: Select Environment`
- `Glubean: Open Last Result JSON`

## Default keybindings

- Re-run last request:
  - macOS: `Cmd+Shift+R`
  - Windows/Linux: `Ctrl+Shift+R`

## High-use settings

- `glubean.glubeanPath`
- `glubean.autoDiscover`
- `glubean.traceHistoryLimit`
- `glubean.taskTimeoutMs`
- `glubean.telemetry.enabled`

For complete tables and descriptions, use [Extension Commands & Settings (Deep Dive)](/extension/reference).

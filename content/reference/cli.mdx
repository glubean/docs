# CLI Quick Reference

## Core commands

```bash
glubean run [target] [options]
glubean init [options]
glubean context [options]
glubean scan [dir]
glubean validate-metadata
```

## Common run options

- `--filter <string>`
- `--pick <string>`
- `--tag <string>`
- `--tag-mode <and|or>`
- `--env-file <path>`
- `--verbose`
- `--emit-full-trace`
- `--result-json [path]`
- `--reporter <format>[:<path>]`

For full option details, use [CLI Command Reference](/cli/reference).

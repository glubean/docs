# Glubean

Code-first API testing that runs the same way in editor and CI.

Glubean is not optimized for one-off API pokes. It is optimized for turning API checks into durable regression assets your team can run on every deploy.

## Product Surfaces

Glubean has four integrated surfaces:

1. **[SDK](/sdk)** (`@glubean/sdk`) for writing tests.
2. **[CLI](/cli)** (`@glubean/cli`) for execution and automation.
3. **[VS Code Extension](/extension/quick-start)** for interactive local workflows.
4. **[Cloud](/cloud)** for run history, team dashboards, and artifact viewing.

## Core Principle

Start in `explore/` for fast iteration, then move stable files into `tests/` for committed regression coverage and CI execution.

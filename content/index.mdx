# Glubean Docs

Welcome to the Glubean documentation. Glubean is a code-first API testing tool that lives in your editor.

## Products

- **[VS Code Extension](/extension/quick-start)** — run tests from the gutter, trace every request, diff between runs
- **[SDK](/sdk/reference)** — TypeScript-first API for writing tests, powered by `@glubean/sdk`
- **[CLI](/cli/commands)** — the engine behind everything: run, init, and manage your tests from the terminal
